在windows上安装jekyll

发表于 2017-01-20 | 分类于 other | 阅读次数:

早些时候用的ubuntu，安装jekyll确实挺简单，可能是终端强大吧。最近工作环境切换到windows，无奈想写博文的时候却发现需要切换到ubuntu，不是那么方便。打算在windows上部署一下jekyll。

软件环境

  • Windows 10 企业版
  • Ruby 2.3.2-x64

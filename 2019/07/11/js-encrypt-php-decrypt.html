注意事项

  1. AES加密位数跟密钥key有关, 以下是密钥位数和加密对应关系
    1. 16 => AES-128
    2. 24 => AES-192
    3. 32 => AES-256
  2. iv是初始化向量. 超过16字节或者不足16字节都会被补足16字节或者截断到16字节。由于AES是块加密，铭文被分割成固定长度的块（一般是16字节长度），所以iv也是16字节。
